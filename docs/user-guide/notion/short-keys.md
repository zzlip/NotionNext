# Notion 快捷键大全

> 迁移自：[Notion快捷键大全](https://docs.tangly1024.com/article/notion-short-key)  
> 官方参考：[Notion Keyboard Shortcuts](https://www.notion.so/help/keyboard-shortcuts)

下文 `cmd/ctrl` 在 Mac 为 **Command**，Windows/Linux 为 **Control**。

## 一、常用 6 个

| 快捷键 | 作用 |
| --- | --- |
| `cmd/ctrl` + `P` | 搜索 / 最近页面 |
| `cmd/ctrl` + `[` | 上一页 |
| `cmd/ctrl` + `]` | 下一页 |
| `cmd/ctrl` + `N` | 新建页面 |
| `cmd/ctrl` + `shift` + `N` 或 `cmd/ctrl`+点击 | 新窗口打开 |
| `cmd/ctrl` + `shift` + `L` | 切换深色模式 |

**Emoji**：输入 `:` + 英文词，如 `:apple` → 🍎

## 二、Markdown

行首输入后按空格：

| 输入 | 效果 |
| --- | --- |
| `#` `##` `###` | 一/二/三级标题 |
| `>` | 折叠列表 |
| `"` | 引用 |
| `[]` | Todo |
| `*` `-` `+` | 无序列表 |
| `1.` | 有序列表 |

行内：`**粗体**`、`*斜体*`、`` `代码` ``、`~删除线~`

## 三、块编辑（节选）

- `Enter` 新行；`shift`+`Enter` 块内换行  
- `cmd/ctrl`+`B/I/U` 粗体/斜体/下划线  
- `cmd/ctrl`+`K` 链接  
- `cmd/ctrl`+`/` 改块类型、颜色  
- `cmd/ctrl`+`D` 复制块  
- `/` 命令菜单（`/h1`、`/todo`、`/image` 等）
