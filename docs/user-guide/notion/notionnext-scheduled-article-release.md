## 使用方法

文章的发布日期字段除了用于显示，还可以用来控制文章合适发布。

例如设置日期为 `2025-01-05 ` 则在此之前访问站点看不到这篇文章。

![image.png](/legacy/582215055fda4711.png)


### 支持精确到时间

在设置日期的选项框中，点击下方的包含时间，即可输入精确到`时:分` 。这样只有在2025年1月5日的14点后才能看到这篇文章。

![image.png](/legacy/e46065f41a1d0423.png)


## 自动下架

你还可以设置这篇文章合适下架，在发布日期的设置框中，点击“结束日期”，即可出现第二行日期设置栏目，在这里填写你需要下架的日期即可。如果不设置，文章则不会自动下架。

![image.png](/legacy/2aafec5e943f4e99.png)
