# Notion-faster 加速访问

> 迁移自：[Notion-faster加速Notion](https://docs.tangly1024.com/article/notion-faster)

大陆访问 Notion 较慢时，可使用社区项目 **Notion-faster**（反向代理，不存储用户数据）：

- 修改 **Hosts**（Windows / Mac / iPad）  
- 或修改 **DNS**（移动端更方便）  

项目会定期探测可用节点。请以项目仓库最新说明为准，注意合规与安全风险。
