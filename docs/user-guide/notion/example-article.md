### 将进酒

李白〔唐代〕

君不见黄河之水天上来，奔流到海不复回。君不见高堂明镜悲白发，朝如青丝暮成雪。人生得意须尽欢，莫使金樽空对月。天生我材必有用，千金散尽还复来。烹羊宰牛且为乐，会须一饮三百杯。岑夫子，丹丘生，将进酒，杯莫停。与君歌一曲，请君为我倾耳听。钟鼓馔玉不足贵，但愿长醉不愿醒。古来圣贤皆寂寞，惟有饮者留其名。陈王昔时宴平乐，斗酒十千恣欢谑。主人何为言少钱，径须沽取对君酌。五花马、千金裘，呼儿将出换美酒，与尔同销万古愁。

### 同步块说明

Notion 支持将不同页面的块进行同步（SyncBlock）。示例段落可来自另一页面并通过 SyncBlock 同步展示。

::: warning
同步块的使用条件是**源页面也要被开放共享**，否则 NotionNext 将无权访问，页面上会被忽略渲染。
:::

## 模板使用说明

部署 NotionNext 时，请复制 Notion 模板并按下列属性创建文章：

| 属性 | 必填 | 说明 |
| --- | --- | --- |
| `title` | 是 | 文章标题 |
| `status` | 是 | 发布状态（仅 `Published` 会展示） |
| `type` | 是 | `Post`（博文）或 `Page`（单页，不出现在列表） |
| `summary` | 否 | 摘要，用于搜索与列表 |
| `date` | 否 | 发布日期（V3.3.9 之前为必填） |
| `category` | 否 | 分类 |
| `tags` | 否 | 标签，不宜过多 |
| `slug` | 否 | 短路径，每篇唯一 |
| `icon` | 否 | 菜单图标（仅 `Page`） |
| `password` | 否 | 文章密码 |
