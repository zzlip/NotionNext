# 用配置调整样式（无需深入 React）

> 迁移自：[NotionNext用配置调整样式](https://docs.tangly1024.com/article/custom-your-style)

适合不熟悉 React/Next/Webpack 的站长：

1. **Notion Config**：`GLOBAL_CSS`、`GLOBAL_JS`  
2. **`public/css/custom.css`**、**`public/js/custom.js`**  
3. **`CUSTOM_EXTERNAL_CSS` / `CUSTOM_EXTERNAL_JS`** 引入外链  
4. **主题 config**：如 `themes/hexo/config.js` 中的开关与文案  
5. **`public/css/theme-<主题名>.css`** 覆盖主题样式

改代码后提交 Git 或在 Vercel Redeploy 生效。
