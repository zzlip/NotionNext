# 交流社群

> 迁移自：[3.社群讨论](https://docs.tangly1024.com/article/chat-community)

## 官方协作（推荐）

NotionNext 已移交 **[notionnext-org](https://github.com/notionnext-org)** 社区维护。**技术问题、文档改进、功能讨论请优先使用 GitHub**，便于搜索与志愿者接手：

| 渠道 | 用途 |
| --- | --- |
| [参与社区](../community-participate.md) | 四步入口总览 |
| [Discussions](https://github.com/notionnext-org/NotionNext/discussions) | 使用问答、想法、文档、开发讨论 |
| [Issues](https://github.com/notionnext-org/NotionNext/issues) | Bug、部署错误、可跟踪任务 |
| [在线文档](https://notionnext.tangly1024.com) | 教程与页底「文档反馈」（Giscus） |

维护者为志愿者，**无商业 SLA**；详见 [MAINTAINERS.md](../../../MAINTAINERS.md)。

## 站长微信群

入群费与规则以源站最新说明为准（曾为付费入群）。**NotionNext / 文档贡献者**可凭已合并 PR 联系小助理申请免付费入群（可选，非参与开发的必要条件）。

技术 BUG 与功能请求：**请勿仅在群内反馈**，请提交 [GitHub Issues](https://github.com/notionnext-org/NotionNext/issues) 或 [Discussions](https://github.com/notionnext-org/NotionNext/discussions)。

## Telegram

面向海外用户的 Notion 站长交流，规则同上：**非官方技术支持渠道**。

## 开发者

无单独「开发者微信群」。参与方式：

1. 阅读 [参与社区](../community-participate.md) 与 [CONTRIBUTING.zh-CN.md](../../../CONTRIBUTING.zh-CN.md)  
2. 提交 PR；合并后可在 Discussion 自荐 [维护者](../../../MAINTAINERS.md) 负责域  

Discord 频道已停止维护。
