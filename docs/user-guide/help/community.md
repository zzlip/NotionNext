# 交流社群

> 迁移自：[3.社群讨论](https://docs.tangly1024.com/article/chat-community)

## 说明

此处为 **Notion 站长交流**，非投资交易群。作者不参与日常群聊，技术问题请走 GitHub Issue。

## 站长微信群

入群费与规则以源站最新说明为准（曾为付费入群；NotionNext / 文档贡献者可联系小助理免付费）。

技术 BUG：请提交 [GitHub Issues](https://github.com/notionnext-org/NotionNext/issues)。

## Telegram

面向海外用户。

## 开发者

无单独开发者群；合并 PR 成为 [Contributors](https://github.com/notionnext-org/NotionNext/graphs/contributors) 后可联系入站长群。

Discord 频道已停止维护。
