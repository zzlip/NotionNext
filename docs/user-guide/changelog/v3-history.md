### 3.12.1 (2023.3.10)
  - 微调simple主题样式
  - 网站图片URL设置
    - 功能简介：
该配置将影响您站点中每张图片URL的格式，可根据实际情况选择您的配置，当您的站点图片无法访问时，可以尝试修改此配置。
    - 配置方式：
在 [blog.config.js](https://github.com/tangly1024/NotionNext/blob/480aa28b21a44124946236377a55225967525a76/blog.config.js#L224) 文件中设置，支持`Notion`和`Amazon`两种类型，**默认是**`**Notion**`)
    - 功能详情及使用场景
      1. **Notion**：图片URL是notion.so的域名，此链接是永久不会挂的，利用的Notion-api；但当图片格式未知，或您的Notion页面没有开启share的情况下，此方法不适用。
        - 参考配置
```JavaScript
IMG_URL_TYPE: process.env.NEXT_PUBLIC_IMG_TYPE || 'Notion',
// 图片前缀将为 https://notion.so/images/xx
```
      1. **Amazon**：图片URL是notion的亚马逊s3图床，即链接为 amazonaws.com的域名。此类URL有24小时有效期，需手动刷新；但不会强制检测图片格式，且您可以不公开分享notion页面。
        - 参考配置
```JavaScript
IMG_URL_TYPE: process.env.NEXT_PUBLIC_IMG_TYPE || 'Amazon',
// 图片前缀将为 https://s3.us-west-2.amazonaws.com/xxx
```
