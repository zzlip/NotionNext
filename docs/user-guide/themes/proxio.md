## 主题特性

- “一人公司“主题，既是个人简历，也是公司介绍。

- 引入了鼠标白点追踪特效

- 引入了lenis网页滚动阻尼动效，页面滚动时会有阻尼感。


## 使用方法

- 将NotionNext当前主题配置为proxio即可

- 主题所有的配置文件可以在 [/themes/proxio/config.js](https://github.com/tangly1024/NotionNext/blob/main/themes/proxio/config.js) 中找到。
