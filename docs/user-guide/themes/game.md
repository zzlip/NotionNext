# Game 主题

> 主题 ID：`game` · 预览：[preview.tangly1024.com/?theme=game](https://preview.tangly1024.com/?theme=game)

## 简介

偏游戏与像素元素装饰。

## 适用场景

游戏像素风装饰

## 启用方式

1. Notion Config 表：`THEME` = `game`
2. 环境变量：`NEXT_PUBLIC_THEME=game`
3. `blog.config.js` 的 `THEME`

## 配置说明

配置文件：[`themes/game/config.js`](../../../themes/game/config.js)  
也可在 **Notion Config** 表中填写同名键（对象/数组用 JSON）。
