# 内置主题说明文档（25 个）

每个主题一篇独立说明：**主题特性**、**启用方式**与 **`themes/<id>/config.js` 配置表** 与代码同步。维护者可运行：

```bash
node scripts/generate-theme-user-docs.mjs
```

从各主题 `config.js` 重新生成配置表（改键名后请执行）。`proxio` / `heo` 含手写章节，脚本只更新 `<!-- theme-config-table -->` 段。
