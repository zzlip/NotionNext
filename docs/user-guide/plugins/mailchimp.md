# Mailchimp 邮件订阅

> 迁移自：[Mailchimp邮件订阅](https://docs.tangly1024.com/article/notion-next-mailchimp)

## 说明

Mailchimp 免费档约 2000 订阅者、每月约 12000 封邮件。部分主题（如 `landing`、`starter`、`proxio`）内置订阅 UI，需先配置 API。

## 环境变量

| 变量名 | 说明 |
| --- | --- |
| `MAILCHIMP_LIST_ID` | 受众列表 ID |
| `MAILCHIMP_API_KEY` | API Key |

主题侧开关示例：`NEXT_PUBLIC_THEME_PROXIO_NEWSLETTER=true`（以各主题 `config.js` 为准）。
