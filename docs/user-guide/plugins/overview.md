# 外部插件总览

> 对应 `conf/widget.config.js`、`conf/animation.config.js`、`conf/plugin.config.js`、`conf/ad.config.js`、`conf/techgrow.config.js`

## 挂件与交互

| 功能 | 配置键 | 文档 |
| --- | --- | --- |
| 主题切换面板 | `THEME_SWITCH` | [themes/overview.md](../themes/overview.md) |
| 音乐播放器 | `MUSIC_PLAYER_*` | [music-player.md](./music-player.md) |
| Live2D 宠物 | `WIDGET_PET` | [features.md](../reference/features.md) |
| Chatbase | `CHATBASE_ID` | 同上 |
| Webwhiz / Dify 客服 | `WEB_WHIZ_*`、`DIFY_*` | 同上 |
| Facebook Page | `FACEBOOK_PAGE_*` | 同上 |
| 剧透隐藏 | `SPOILER_TEXT_TAG` | 同上 |

## 页面动效

| 功能 | 配置键 |
| --- | --- |
| 点击烟花 | `FIREWORKS` |
| 鼠标跟随 | `MOUSE_FOLLOW` |
| 樱花 / 星空 / 彩带 | `SAKURA`、`STARRY_SKY`、`RIBBON` 等 |
