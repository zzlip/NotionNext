# 外部插件说明

> 迁移自：[外部插件说明](https://docs.tangly1024.com/article/notion-next-plugins)
> 发布日期：2023-6-27
> 最后编辑：2024-9-7
> 原栏目：🧷 外部扩展

NotionNext 预支持了一些第三方功能脚本插件。除了评论插件，还支持搜索、广告、背景动效、宠物挂件、音乐播放器、邮件订阅和在线客服等扩展。

开启插件的方法通常是在 `blog.config.js` 或 `conf/*.config.js` 中找到对应配置，或者在部署平台环境变量中添加同名配置项。不同插件需要的密钥、脚本地址或服务端地址不同，请进入对应教程查看。
