# Valine 评论

> 迁移自：[Valine](https://docs.tangly1024.com/article/notionnext-valine)

基于 LeanCloud。与 Waline 数据可互通。

## NotionNext 配置

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_VALINE_ID` | LeanCloud App ID |
| `NEXT_PUBLIC_VALINE_KEY` | App Key |
| `NEXT_PUBLIC_VALINE_SERVER_URLS` | 国内自定义域名时填 `https://xxx` 请求域名 |
| `NEXT_PUBLIC_VALINE_PLACEHOLDER` | 占位文案 |

LeanCloud 控制台 → 设置 → 应用 Key 获取 ID/Key；Request 域名需加 `https://` 前缀。
