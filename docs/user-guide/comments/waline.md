# Waline 评论

> 迁移自：[Waline](https://docs.tangly1024.com/article/notion-next-waline)

Waline 由 Valine 演进，数据结构兼容，可共用 LeanCloud 或自建 MySQL / PostgreSQL 等。

## NotionNext 配置

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_WALINE_SERVER_URL` | Waline 服务完整 URL，如 `https://waline.example.com` |

部署方式：Vercel、Railway、Docker 等，见 [waline.js.org](https://waline.js.org/guide/get-started.html)。
