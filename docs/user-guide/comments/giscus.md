# Giscus 评论

> 迁移自：[Giscus](https://docs.tangly1024.com/article/notion-next-giscus)

基于 GitHub Discussions。在 [giscus.app](https://giscus.app/) 生成配置后填入：

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_COMMENT_GISCUS_REPO` | 仓库名 `owner/repo` |
| `NEXT_PUBLIC_COMMENT_GISCUS_REPO_ID` | Repo ID |
| `NEXT_PUBLIC_COMMENT_GISCUS_CATEGORY` | 分类名 |
| `NEXT_PUBLIC_COMMENT_GISCUS_CATEGORY_ID` | 分类 ID |

更多选项见 `conf/comment.config.js` 中 `COMMENT_GISCUS_*`。
