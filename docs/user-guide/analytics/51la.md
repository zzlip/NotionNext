# 51LA 站点统计

> 迁移自：[51LA站点统计](https://docs.tangly1024.com/article/notion-next-51-la)

## 环境变量

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_ANALYTICS_51LA_ID` | 统计 ID |
| `NEXT_PUBLIC_ANALYTICS_51LA_CK` | CK 参数 |

在 [51.la](https://www.51.la/) 注册后，于「手动安装」中获取上述两项。
