# Microsoft Clarity

> 迁移自：[Clarity站点统计](https://docs.tangly1024.com/article/notion-next-clarity)

## 环境变量

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_CLARITY_ID` | Clarity 项目 ID（十位字母数字） |

在 [clarity.microsoft.com](https://clarity.microsoft.com/) 创建项目后，从嵌入脚本中复制 ID 部分即可。
