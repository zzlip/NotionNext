# Umami 统计

> 迁移自：[UMAMI统计](https://docs.tangly1024.com/article/notion-next-umami)

## 环境变量

| 变量名 | 说明 |
| --- | --- |
| `NEXT_PUBLIC_UMAMI_HOST` | 脚本地址，默认 `https://cloud.umami.is/script.js` |
| `NEXT_PUBLIC_UMAMI_ID` | 网站 ID |

支持 [Umami Cloud](https://umami.is/) 或自建。
