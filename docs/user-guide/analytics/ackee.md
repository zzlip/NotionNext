# Ackee 自托管统计

> 迁移自：[Ackee站点统计](https://docs.tangly1024.com/article/notion-next-ackee)

## 环境变量

| 变量名 | 示例 |
| --- | --- |
| `NEXT_PUBLIC_ANALYTICS_ACKEE_DATA_SERVER` | `https://ackee.example.com`（勿尾斜杠） |
| `NEXT_PUBLIC_ANALYTICS_ACKEE_TRACKER` | `https://ackee.example.com/tracker.js` |
| `NEXT_PUBLIC_ANALYTICS_ACKEE_DOMAIN_ID` | 站点 UUID |

可用 Vercel + MongoDB 等免费方案自建，见 Ackee 官方部署文档。
