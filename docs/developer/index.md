# NotionNext 开发文档

NotionNext 文档不再把“站长教程”和“开发者文档”做成两个隔离区域。每个使用者都可以先完成部署和配置，再逐步理解运行原理、修改主题、提交 PR。

如果你刚开始搭建网站，先读 [从这里开始](../user-guide/start-here.md)。当你想改样式、改主题、排查问题或参与维护时，继续阅读本页下面的开发文档。
